    flickr.activity.userComments request.

    Returns a list of recent activity on photos commented on by the calling user. Do not poll this method more than once an hour.

    Arguments

    per_page

    Number of items to return per page. If this argument is omitted, it defaults to 10. The maximum allowed value is 50.

    page

    The page of results to return. If this argument is omitted, it defaults to 1.

    flickr.activity.userPhotos request.

    Returns a list of recent activity on photos belonging to the calling user. Do not poll this method more than once an hour.

    Arguments

    timeframe

    The timeframe in which to return updates for. This can be specified in days ('2d') or hours ('4h'). The default behavoir is to return changes since the beginning of the previous user session.

    per_page

    Number of items to return per page. If this argument is omitted, it defaults to 10. The maximum allowed value is 50.

    page

    The page of results to return. If this argument is omitted, it defaults to 1.
